# Data-Driven-Smart-food-analytics-and-advisor
An intelligent AI-powered web application designed to simplify food tracking and provide personalized nutrition insights using data analytics and machine learning.
This system allows users to log their meals using text descriptions or food images, automatically detects food items, estimates nutritional values, and generates actionable health insights based on individual profiles and dietary goals.

1. AI-Based Food Detection
- Detects food items from images or text input using AI models
- Supports diverse cuisines including complex mixed dishes

2. Automated Nutrition Analysis
-Calculates calories, protein, carbohydrates, fats, and fiber
-Real-time updates with adjustable portion sizes

3. Personalized AI Insights
Generates health scores for meals
Provides suggestions, benefits, and warnings based on user profile

4.Interactive Analytics Dashboard
Visualizes calorie trends and macro distribution
Daily, weekly, and monthly analysis using charts

5. Smart Notifications
Alerts when calorie limits are exceeded
Daily nutrition summary via email

6. User Profile Management
Personalized recommendations based on BMI, goals, and activity level

Tech Stack
-Frontend & UI: Streamlit
-Backend & Logic: Python
-Database: MongoDB Atlas
-AI Integration: Google Gemini API
-Visualization: Plotly
-Authentication: Bcrypt
-Deployment Ready: Environment-based configuration (.env)

How It Works
-User logs in and sets up a health profile
-Inputs meal via image or text
-AI detects food items and estimates nutrition
-User adjusts quantities if needed
-System calculates total nutrition and generates insights
-Data is stored and visualized in analytics dashboard
Use Cases
Personal diet and calorie tracking
Fitness and weight management
Preventive healthcare monitoring
Nutrition awareness and lifestyle improvement

 Highlights
